# Weather Station API
## Requirements

- Docker & Docker-Compose : latest
- MongoDB : latest
- Python 3.7
- Required libraries (requirements.txt)




## Instructions
To run the database and API containers locate installation folder and execute the following line in a terminal:
```
docker-compose up
```
